# Tech Stack

The **Schematic Sync Portal** is a high-security TUI application built with:

- **Runtime**: [Bun](https://bun.sh/)
- **Frontend**: [React](https://react.dev/) (TUI)
- **UI Framework**: [@opentui/core](https://www.npmjs.com/package/@opentui/core) & [@opentui/react](https://www.npmjs.com/package/@opentui/react)
- **Backend Execution**: [Rclone](https://rclone.org/) CLI Backend
- **Configuration**: JSON-based (`~/.config/schem-sync-portal/config.json`)
- **Package Manager**: Bun
